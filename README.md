# capture カーソルもキャプチャする
### 1) WIN + テンキーののプラスキーで、拡大鏡を起動
### 2) CTRL + ALT + D で固定ビュー
### 3) 上下半分づつで、カーソルの必要な画面が上半分に表示されるようにする
### 4) PrintScreen で全体をキャプチャ
### 5) CTRL + ALT + F で全体ビューに戻す
### 6) mspaint に貼り付けて必要な部分を取り出す
